NAEI Multi-Group Pollutant Viewer

🧭 How to Use the NAEI Multi-Group Pollutant Viewer

The NAEI Multi-Group Pollutant Viewer is built from data published by the National Atmospheric Emissions Inventory (NAEI), combining one or more sources and fuel types into groups to make emissions easier to explore and compare.

1️⃣ Select a Pollutant

Use the Pollutant dropdown at the top to choose which pollutant you want to view (e.g., PM2.5, CO₂, NOx, etc.).
Once selected, the chart and group options update automatically.

2️⃣ Choose Emission Groups

Click “+ Add Group” to add a new group comparison (e.g., Road Transport, Power Stations, Domestic Combustion).
You can add up to 10 groups to compare at once.
To remove a group, click “− Remove Group” next to it (this appears when two or more groups are selected).

💡 Tip: You can re-order the group boxes by clicking and dragging the ⠿ handle next to each group — the chart updates automatically.

3️⃣ Set the Year Range

Use the Start Year and End Year dropdowns to focus on a particular time period.
The chart will automatically refresh to show data for the chosen range.

4️⃣ View and Interact with the Chart

Once you’ve chosen your pollutant, groups, and years, the interactive line chart will appear.
Hover over any point on the chart to see the exact emission value for that year.
The Y-axis shows total emissions, including units (e.g., kilotonnes).
Each group is shown in a unique colour.

5️⃣ Use the Legend to Toggle Lines

Above the chart, the legend lists all groups and their corresponding colours.
Click a group name in the legend to hide or show that group’s line on the chart.
Faded items indicate hidden or unavailable data.
(Groups labelled “(No data available)” are included for reference but contain no plotted values.)

6️⃣ Smooth or Straight Lines

Click the “🚫 Disable Smoothing” button to switch to straight (non-curved) lines.
Click again (the button will change to “✅ Enable Smoothing”) to re-enable smooth, curved lines.

7️⃣ Download Options

You can export both the chart and the data for your own analysis or reporting.

📊 Download Chart as PNG
Click “⬇️ Download Chart as PNG” to save the chart as an image.
The export includes:

  • The chart
  • The custom colour legend
  • The pollutant name and emission unit
  • A small Defra/DESNZ footer and the CIC logo

📄 Download Data
CSV: Click “📄 Download Data – CSV” for a spreadsheet-compatible text file.
Excel: Click “📊 Download Data – Excel” for a formatted Excel workbook.
Each data file includes:

  • A first line showing the pollutant name and emission unit
  • A table with groups as rows and years as columns
  • A footer row with the download timestamp

8️⃣ View Group Details

Use the “📘 See Group Info” section below the group selector to view details of how each group is built.
This table shows the Source Names and Fuel Types, from the NAEI data, that have been combined to create each group.
If a group includes all sources or all fuel types, these will be shown as “All Sources” or “All Fuel Types” respectively.
