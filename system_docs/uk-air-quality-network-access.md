# UK Air-Quality Networks & Data Access

_Last reviewed: December 2025._

Each row lists a network that operates monitors within the United Kingdom, whether statutory, local-authority, or citizen/community. Columns cover the network label, whether its footprint is UK-only or multi-country, and the primary access method with a pointer to documentation or download endpoints.

| Network | Coverage Scope | Access Type & Docs |
| --- | --- | --- |
| DEFRA UK-AIR / Automatic Urban and Rural Network (AURN) | UK-only | REST + CSV downloads via UK-AIR API (`https://uk-air.defra.gov.uk/data/`). |
| Air Quality England (Ricardo AQC local-authority portal) | UK-only (multiple English councils) | CSV/JSON station feeds and ArcGIS REST layers at `https://www.airqualityengland.co.uk/site/download`. |
| Scottish Air Quality Network (`scottishairquality.scot`) | UK-only (Scotland) | JSON + CSV downloads (`https://www.scottishairquality.scot/data/data-downloads`). |
| Wales Air Quality Network (`airquality.gov.wales`) | UK-only (Wales) | CSV/JSON API plus ArcGIS feature layers (`https://airquality.gov.wales/data`). |
| Northern Ireland Air (`airqualityni.co.uk`) | UK-only (Northern Ireland) | CSV downloads and JSON feed (`https://www.airqualityni.co.uk/air-quality-data`). |
| London Air Quality Network (LAQN / LondonAir) | UK-only (Greater London) | REST/JSON & CSV API documented at `https://www.londonair.org.uk/londonair/guide/api.aspx`. |
| Breathe London Network (AQMesh pods) | UK-only (Greater London) | REST/JSON API + bulk CSV at `https://www.breathelondon.org/api`. |
| Greater Manchester Combined Authority AQ Network (Clean Air GM) | UK-only (Greater Manchester) | ArcGIS REST & CSV endpoints linked from `https://cleanairgm.com/data/`. |
| Birmingham ENVIZ / West Midlands local authority network | UK-only (West Midlands) | ArcGIS REST feature service + CSV exports published via `https://www.birmingham.gov.uk/info/20076/pollution/1763/air_quality/4`. |
| EEA e-Reporting (UK submissions) | Multi-country | REST/JSON & CSV via EEA AQ portal (`https://aqportal.discomap.eea.europa.eu/`). |
| OpenAQ (UK sources) | Multi-country | REST API with provenance metadata (`https://docs.openaq.org/`). |
| Sensor.Community (UK contributors) | Multi-country | JSON API + CSV dumps documented at `https://sensor.community/en/docs/`. |
| PurpleAir (UK deployments) | Multi-country | Tokened REST + WebSocket API (`https://api.purpleair.com/`). |
| Airly municipal deployments in UK | Multi-country | REST/JSON developer API (`https://developer.airly.org/`). |
| uRADMonitor (devices installed in UK) | Multi-country | REST JSON per device (`https://www.uradmonitor.com/documentation/`). |
